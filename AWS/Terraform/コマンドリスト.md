# Terraformコマンド
## terraform init
### 概要
Terraform実行環境の初期化

### 主な処理
| 処理 | 内容 |
| --- | --- |
| Provider取得 | AWS ProviderなどDL |
| Module取得 | module source取得 |
| Backend初期化 | S3 backend接続 |
| .terraform 作成 | 作業ディレクトリ生成 |
| lock file作成 | version固定 |

### 実行タイミング
- 初回
- provider変更
- module変更
- backend変更

### 生成されるもの
.terraform/

.terraform.lock.hcl

## terraform plan
### 概要
現在AWS上に存在するリソースと、tfファイルとの差分を比較

意図している変更がされるかをapply前に確認
例）Resources: 0 added, 0 changed, 0 destroyed.

## terraform apply
### 概要
planで確認した変更を実際に実行

## terraform fmt
### 概要
コード整形

### 処理
- インデント修正
- 空白修正
- 改行統一
- 文法チェック
